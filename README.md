# Human Benchmark Reaction Time Test Clone

A React.js clone of the popular Human Benchmark reaction time test.

## Features
- Reaction time measurement
- Statistics tracking (average, median, best)
- Visual chart of reaction times
- Save scores functionality
- Responsive design
